# SUDOKU_GRILLAS_CODIFICADO_9X9 🔠

Este repositorio contiene el sistema especializado en la representación visual y codificación de grillas de Sudoku 9x9, desarrollado en Visual Basic 6.0.

## 🚀 Descripción

`SUDOKU_GRILLAS_CODIFICADO_9X9` es una herramienta avanzada que permite la visualización dinámica de soluciones de Sudoku vinculándolas directamente con sus identificadores o códigos únicos. El proyecto se centra en la "Grilla Codificada", una interfaz que permite al investigador no solo ver los números, sino entender la estructura de la plantilla (*template*) que subyace en cada solución.

Es un componente clave para la auditoría y validación visual de los algoritmos de codificación generados en el ecosistema de investigación de Sudokus.

## 🛠️ Especificaciones Técnicas

- **Lenguaje:** Visual Basic 6.0.
- **Arquitectura:** Basada en componentes de interfaz (ActiveX/Custom Controls) para el renderizado de celdas.
- **Funcionalidades Clave:**
  - Visualización de grillas con soporte para resaltado de patrones de codificación.
  - Vinculación en tiempo real entre la disposición numérica y su código identificador.
  - Capacidad de importar y exportar estados de grilla codificados.
  - Interfaz optimizada para el análisis de simetrías en matrices 9x9.

## 📂 Estructura del Repositorio

Tras la reestructuración profesional:
- `/src`: Formularios de visualización, clases de codificación y módulos de soporte.
- `/assets`: Iconos y recursos visuales para la interfaz de las grillas.
- `/docs`: Documentación sobre la metodología de codificación visual.
